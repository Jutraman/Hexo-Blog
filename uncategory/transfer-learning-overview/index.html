Transfer Learning 迁移学习

Why

  • 大数据与少标注的矛盾
  • 大数据与弱计算的矛盾
  • 有限数据与模型泛化的矛盾
  • 普适化与个性化需求的矛盾
  • 特定需求

Where

特征空间

  • 同构迁移学习
  • 异构迁移学习

目标域

  • 有监督迁移学习
  • 半监督迁移学习
  • 无监督迁移学习

学习方法

  • 基于实例(Priority: 1)
  • 基于模型(Priority: 3)
  • 基于特征(Priority: 2)
  • 基于关系(Priority: 0)

状态

  • 离线
  • 在线

What

非官方定义

迁移学习可以利用数据、任务或者模型之间的相似性，将在旧领域学习过的模型和知识应用于新领域。

基本概念

  • 源域 Source Domain=Ds
  • 目标域 Target Domain=Dt
  • 样本特征空间Xs,Xt
  • 样本标签特征空间Ys,Yt
  • 样本概率分布 Ps(x,y),Pt(x,y)

数学表达

  • Xs != Xt
  • Ys != Yt
  • Ps != Pt

本质上是Ds选择
